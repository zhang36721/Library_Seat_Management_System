# 串口调试工具

本目录用于存放串口调试相关的工具和日志。

## 工具（待添加）

- 串口调试助手（Windows/Mac/Linux）
- 串口日志分析脚本
- 数据解析工具

## 调试说明

### 座位节点调试
1. 连接 ST-Link 和串口模块
2. 打开串口调试助手
3. 设置波特率 9600
4. 观察数据输出

### 主控节点调试
1. 连接 ST-Link 和串口模块
2. 打开串口调试助手
3. 设置波特率 115200（与 ESP32S3 通信）
4. 观察数据收发

### ESP32S3 调试
1. 连接 USB 线
2. 打开串口监视器
3. 设置波特率 115200
4. 查看 WiFi 连接和数据上传日志
