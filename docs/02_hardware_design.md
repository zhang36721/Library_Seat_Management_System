# 硬件设计说明

## 1. 座位检测节点

### 1.1 核心组件
- **MCU**: STM32F103C8T6
- **无线模块**: CC2530 ZigBee 模块
- **传感器**: 红外热释电传感器 / 压力传感器
- **电源**: 3.7V 锂电池 + 充电管理

### 1.2 设计要点
- 低功耗设计，支持睡眠模式
- 传感器定时唤醒检测
- ZigBee 自组网通信

## 2. 主控节点

### 2.1 核心组件
- **MCU**: STM32F103C8T6 / STM32F103ZET6（根据节点数量选择）
- **无线模块**: CC2530 ZigBee 协调器
- **通信接口**: UART 转 ESP32S3
- **电源**: 5V USB 供电

### 2.2 设计要点
- ZigBee 协调器模式
- 多节点数据汇聚
- 数据缓存和重传机制

## 3. WiFi 网关

### 3.1 核心组件
- **MCU**: ESP32S3
- **通信接口**: UART 转主控 STM32
- **网络**: WiFi 2.4GHz
- **电源**: 5V USB 供电

### 3.2 设计要点
- WiFi 自动重连
- HTTP/HTTPS 数据上传
- 断网数据缓存
