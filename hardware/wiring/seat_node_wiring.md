# 座位检测节点接线说明

## 1. 组件清单

| 组件 | 数量 | 备注 |
|------|------|------|
| STM32F103C8T6 最小系统板 | 1 | 或自定义 PCB |
| CC2530 ZigBee 模块 | 1 | 终端节点 |
| 红外热释电传感器 | 1 | 或压力传感器 |
| 3.7V 锂电池 | 1 | 可选，用于独立供电 |
| TP4056 充电模块 | 1 | 可选 |

## 2. 接线图

### 2.1 STM32 与 ZigBee 模块

| STM32 引脚 | ZigBee 引脚 | 说明 |
|------------|-------------|------|
| PA9 (USART1_TX) | RXD | 串口发送 |
| PA10 (USART1_RX) | TXD | 串口接收 |
| 3.3V | VCC | 电源 |
| GND | GND | 地 |

### 2.2 STM32 与传感器

| STM32 引脚 | 传感器引脚 | 说明 |
|------------|------------|------|
| PA0 (ADC1_IN0) | OUT | 模拟输入（如使用模拟传感器）|
| 3.3V | VCC | 电源 |
| GND | GND | 地 |

## 3. 注意事项

1. ZigBee 模块使用 3.3V 供电，不要接 5V
2. 串口通信波特率：9600
3. 传感器根据实际型号调整接线
4. 低功耗设计建议添加电源开关

## 4. 调试接口

| 接口 | 用途 |
|------|------|
| SWDIO | 调试和烧录 |
| SWCLK | 调试时钟 |
| GND | 地 |
| 3.3V | 可选供电 |
