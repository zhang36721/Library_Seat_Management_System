# ESP32S3 WiFi 网关接线说明

## 1. 组件清单

| 组件 | 数量 | 备注 |
|------|------|------|
| ESP32S3 开发板 | 1 | 如 ESP32S3 DevKit |
| USB 转 TTL 模块 | 1 | 如果开发板自带 USB 转串口则不需要 |
| 5V 电源适配器 | 1 | USB 供电 |

## 2. 接线图

### 2.1 ESP32S3 与 USB 转 TTL 模块（如需要）

| ESP32S3 引脚 | USB-TTL 引脚 | 说明 |
|--------------|--------------|------|
| GPIO44 (RX) | TXD | 串口接收（从 STM32） |
| GPIO43 (TX) | RXD | 串口发送（到 STM32） |
| GND | GND | 共地 |
| 5V | 5V | 电源（如开发板需要外部供电） |

### 2.2 ESP32S3 与主控 STM32

| ESP32S3 引脚 | STM32 引脚 | 说明 |
|--------------|------------|------|
| GPIO44 (RX) | PA2 (USART2_TX) | 接收 STM32 数据 |
| GPIO43 (TX) | PA3 (USART2_RX) | 发送数据到 STM32 |
| GND | GND | 共地 |

## 3. WiFi 配置

### 3.1 网络连接
- **WiFi 频段**: 2.4GHz
- **安全协议**: WPA2/WPA3
- **IP 获取方式**: DHCP 或静态 IP

### 3.2 服务器连接
- **协议**: HTTP/HTTPS
- **端口**: 80/443
- **数据格式**: JSON

## 4. 注意事项

1. ESP32S3 使用 3.3V 电平，确保与外部模块电平匹配
2. 串口通信波特率：115200
3. 确保 WiFi 信号强度足够
4. 建议添加看门狗防止死机

## 5. 调试接口

| 接口 | 用途 |
|------|------|
| USB | 烧录和调试 |
| EN | 使能引脚（高电平有效） |
| GPIO0 |  boot 模式选择（拉低进入下载模式） |
