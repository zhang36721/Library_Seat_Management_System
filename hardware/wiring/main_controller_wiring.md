# 主控节点接线说明

## 1. 组件清单

| 组件 | 数量 | 备注 |
|------|------|------|
| STM32F103C8T6 最小系统板 | 1 | 或 STM32F103ZET6（根据节点数量选择） |
| CC2530 ZigBee 模块 | 1 | 协调器节点 |
| USB 转 TTL 模块 | 1 | 用于与 ESP32S3 通信 |
| 5V 电源适配器 | 1 | USB 供电 |

## 2. 接线图

### 2.1 STM32 与 ZigBee 模块（协调器）

| STM32 引脚 | ZigBee 引脚 | 说明 |
|------------|-------------|------|
| PA9 (USART1_TX) | RXD | 串口发送 |
| PA10 (USART1_RX) | TXD | 串口接收 |
| 3.3V | VCC | 电源 |
| GND | GND | 地 |

### 2.2 STM32 与 USB 转 TTL 模块

| STM32 引脚 | USB-TTL 引脚 | 说明 |
|------------|--------------|------|
| PA2 (USART2_TX) | RXD | 串口发送（到 ESP32S3） |
| PA3 (USART2_RX) | TXD | 串口接收（从 ESP32S3） |
| GND | GND | 共地 |

### 2.3 USB 转 TTL 与 ESP32S3

| USB-TTL 引脚 | ESP32S3 引脚 | 说明 |
|--------------|--------------|------|
| TXD | RX (GPIO44) | 数据发送 |
| RXD | TX (GPIO43) | 数据接收 |
| GND | GND | 共地 |
| 5V | VIN | 电源（如需要） |

## 3. 注意事项

1. ZigBee 协调器需要配置为协调器模式
2. 确保所有模块共地
3. USB 转 TTL 模块选择 3.3V 电平
4. 串口通信波特率：115200（与 ESP32S3）

## 4. 调试接口

| 接口 | 用途 |
|------|------|
| SWDIO | 调试和烧录 |
| SWCLK | 调试时钟 |
| GND | 地 |
| 3.3V | 可选供电 |
